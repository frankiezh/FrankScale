FrankScale is the canonical product codebase. It is a mixed Rust + TypeScript monorepo for the FrankScale/Vibe Kanban product, with backend workspace crates under `crates/`, shared generated types under `shared/`, and React frontends under `packages/local-web`, `packages/remote-web`, and `packages/web-core`. Product-core changes must remain tenant-agnostic; tenant-specific rollout/business logic belongs in config, deployment docs, integration layers, or admin-managed data, not shared product logic. Key structure: `crates/server`, `crates/remote`, `crates/db`, `crates/services`, `crates/executors`, `crates/api-types`; frontend shell in `packages/local-web/src`; shared UI/app logic in `packages/web-core/src`; packaged/local assets in `assets/`, `dev_assets_seed/`, `dev_assets/`; helper scripts in `scripts/`; docs in `docs/`.
