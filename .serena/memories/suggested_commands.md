Core commands for FrankScale development on Linux: `pnpm i` to install JS dependencies; `pnpm run dev` to run web app + backend with auto-assigned ports; `pnpm run backend:dev:watch` for backend watch mode; `pnpm run local-web:dev` for the local web app; `pnpm run check` for frontend + backend checks; `pnpm run backend:check` for backend Rust workspace checks; `cargo test --workspace` for Rust tests; `pnpm run generate-types` and `pnpm run remote:generate-types` to regenerate TS types from Rust; `pnpm run prepare-db` and `pnpm run remote:prepare-db` for SQLx prep; `pnpm run format` for formatting; `pnpm run lint` for linting; `pnpm run build:npx` then `pnpm pack` in `npx-cli/` for NPX packaging. General shell commands commonly useful here: `git status`, `git diff`, `git log --oneline`, `rg`, `rg --files`, `ls`, `find`, `sed -n`, `pwd`.
